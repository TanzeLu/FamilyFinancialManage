
it's a simple application ,connecting with mysql database,programmed in java swing.

you can check its interfaces in the directory "FamilyFinancialManage/images".


HOW TO RUN?
-------------------------------------
to run this project,you can import "FinancialManage" and "FinancialSystem2" into the Eclipse.

jdk 1.8 and tomcat9 is requied to be installed in your Eclipse before you start this project.

if all is prepared well,you can start this project by steps showed in follows:

1.run "FinancialManage" in server "Tomcat9"

2.run the "Login.java" or "Regiter.java" in "FinancialSystem2",you should register firstly if you have not logined before.
